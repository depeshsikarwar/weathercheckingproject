# 🌦️ Weather App (HTML, CSS, JavaScript + API)

## 📌 Overview

This is a simple and responsive **Weather Checking Web App** built using **HTML, CSS, and JavaScript**. It fetches real-time weather data from a weather API and displays information like temperature, humidity, wind speed, and weather conditions for any city entered by the user.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays current temperature
* 💧 Shows humidity levels
* 🌬️ Wind speed information
* 🌤️ Weather condition (Clear, Clouds, Rain, etc.)
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Functionality and API handling
* **Weather API** – Fetch real-time weather data (e.g., OpenWeatherMap API)

## ⚙️ How It Works

1. User enters a city name
2. JavaScript sends a request to the weather API
3. API returns weather data in JSON format
4. Data is displayed on the UI

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter a city name and check the weather
